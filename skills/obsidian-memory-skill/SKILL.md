# Obsidian Memory Skill

## Когда использовать
- сохранить мысль в inbox
- записать важное в daily
- обновить MEMORY.md
- связать заметку с MOC
- не потерять важный контекст из чата

## Правила
- не сохранять всё подряд
- сначала определить тип информации
- daily notes не должны быть сиротами
- важные решения связывать с moc-decisions
- новые темы сначала вести через moc-triage

## Логика
- текущий необработанный захват -> inbox
- дневной контекст -> knowledge/memory/daily/
- долгосрочно важное -> MEMORY.md
- важные решения -> maps/moc-decisions.md
- новая тема без ясного домена -> maps/moc-triage.md
