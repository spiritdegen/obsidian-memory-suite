# Memory Hygiene

## Что проверять
- orphan notes
- duplicate titles
- notes without tags
- stale links
- merge candidates
- hot / warm / cold tiers

## Правила
- не удалять автоматически без подтверждения
- сначала показать проблему, потом предложить действие
- использовать canonical tags
- weekly audit проводить регулярно

## Tiers
- hot: 0-7 дней
- warm: 8-30 дней
- cold: 30+ дней
